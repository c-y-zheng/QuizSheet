# QuizSheet

📝 **一款通用、可离线使用的刷题应用，将你的 Excel 表格秒变交互式练习题库。**

只需上传一个 `.xlsx` 题库文件，即可立即获得简洁、响应式的答题界面——适用于学生备考、企业培训、竞赛练习等各类场景。**所有处理均在浏览器中完成：无需后端，数据不出本地。**

## ✨ 核心特性

- **完全离线可用**：首次加载后无需网络，适合内网或安全隔离环境
- **支持常见题型**：单选题、多选题、判断题
- **智能答案解析**：兼容 `A`、`AB`、`A,B,C`、`正确`、`错误`、`√`、`×` 等多种答案格式
- **安全可靠**：所有题目与选项内容自动 HTML 转义，有效防止 XSS 攻击
- **响应式界面**：适配电脑、平板设备（手机建议横屏使用）
- **实时状态反馈**：题号列表通过颜色区分“未答 / 答对 / 答错”
- **键盘友好**：支持 Tab + Enter 快速切换题目
- **零安装部署**：只需打开 `index.html` 文件即可使用

## 📁 使用方法

### 1. 准备题库文件
创建一个 Excel 文件（`.xlsx`），包含以下列（列名必须严格匹配）：

| 列名 | 说明 |
|------|------|
| `题目` | 题干内容 |
| `题型` | 如“单选题”、“多选题”、“判断题”（需包含“单选”“多选”或“判断”关键词） |
| `答案` | 正确答案：<br>• 单选：`A`<br>• 多选：`AB` 或 `A,B,C`<br>• 判断：`正确`、`错误`、`√`、`×` 等 |
| `A`, `B`, `C`, `D`… | 各选项文本（判断题可留空，系统会自动生成“正确/错误”） |

> 💡 **提示**：判断题若未填写 A/B 列，程序将自动补全为“正确”和“错误”。

### 2. 开始刷题
1. 从本仓库下载 [`index.html`](./index.html) 文件  
2. 用浏览器（Chrome、Edge、Firefox 或 Safari）打开该文件  
3. 点击 **“加载题库”** 按钮，选择你的 `.xlsx` 文件  
4. 开始答题！系统会即时反馈正误，完成后显示总分。

> ⚠️ **注意**：首次使用需联网（用于加载 Excel 解析库），之后可完全离线运行。

## 🔒 隐私与安全
- **数据绝不外传**：所有操作在本地浏览器内存中完成
- **防注入保护**：用户内容渲染前自动转义，杜绝脚本执行风险
- **文件沙箱读取**：题库仅在当前页面使用，不会保存或上传

## 📱 兼容性
- ✅ Chrome / Edge / Firefox / Safari（最新版）
- ✅ 电脑与平板设备（手机建议横屏以获得更好体验）

## 🛠 技术栈
- 纯 HTML / CSS / JavaScript（无框架、零依赖）
- [SheetJS (xlsx)](https://sheetjs.com/)：用于解析 Excel 文件（通过 CDN 加载）
